# Add Folder Structure

```
react-template/
├── public/                 # Static assets (favicons, robots.txt)
├── src/
│   ├── assets/             # Global images, fonts, svgs
│   ├── components/         # Shared UI Components (atomic/dumb)
│   │   ├── ui/             # Generic UI from ShadCN (Buttons, Inputs, Modals)
│   │   ├── layout/         # Layout components (Sidebar, Navbar)
│   │   └── ErrorBoundary.tsx
│   ├── features/           # Feature-Based Modules (The core logic)
│   │   ├── auth/           # Example Feature: Authentication
│   │   │   ├── components/ # Components specific to Auth (LoginForm)
│   │   │   ├── api/        # TanStack Query hooks (useLogin, useUser)
│   │   │   ├── types/      # TS interfaces for Auth
│   │   │   ├── routes/     # Route definitions for this feature
│   │   │   └── index.ts    # Public API (exports for other features)
│   │   └── landing/        # Example Feature: landing
│   ├── hooks/              # Global hooks (useTheme, useMediaQuery)
│   ├── lib/                # Library configurations (Axios, QueryClient)
│   │   ├── axios.ts
│   │   ├── query-client.ts
│   │   └── utils.ts        # CN (Tailwind class merger) & formatters
│   ├── routes/             # App-wide routing configuration
│   │   ├── AppRoutes.tsx   # Main router component
│   │   └── protected-route.tsx
│   ├── store/              # Global state (Zustand/Context) - if needed
│   ├── types/              # Global TS types (User, ApiResponse)
│   ├── App.tsx             # Root component (Providers setup)
│   ├── main.tsx            # Entry point
│   └── index.css           # Tailwind directives & global styles
├── .env                    # Environment variables
├── index.html              # Vite entry
├── tsconfig.json           # TypeScript config
└── vite.config.ts          # Vite config
```
